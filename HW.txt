1) HTTP is an application- layer protocol for transmitting hypermedia documents, such as HTML

2)Uniform resource locator which is the address of a web page

3)The Doman Name System translastes human readable domain names to machine readable IP addresses

4)A query string is a part of the URL that assigns values to specified parameters. 

5)Post: Used to "create" new resources on the webpage
    Get: Used to "read" (or retrieve) a representation of a resource.

6)An HTTP request is made by a client, to a named host, which is located on a server. Used to access a resource on the server.

7)An HTTP response is made by a server to a client. 

8)An HTTP header is a field of an HTTP request or response that passes additional context and metadata about the request or response.Examples are cookie requests and content languages

9)Usually it would be a request to access the info from a server to display the webpage.
